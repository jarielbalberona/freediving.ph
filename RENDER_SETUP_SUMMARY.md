## 🚀 How to Deploy

### Option 1: Blueprint Deployment (Recommended)
```bash
# 1. Commit all changes
git add .
git commit -m "Add Render deployment configuration"
git push origin main

# 2. Go to Render Dashboard
# 3. Create New Blueprint
# 4. Connect your GitHub repo
# 5. Render will auto-detect render.yaml
# 6. Deploy all services at once
```

### Option 2: Individual Service Deployment
1. **Deploy Database first** (PostgreSQL service)
2. **Deploy API** (Web service, root directory: `api`)
3. **Deploy Frontend** (Web service, root directory: `app`)

## 🔐 Environment Variables Required

### Critical Variables (Must Set)
```
CLERK_SECRET_KEY=sk_test_...

AWS_ACCESS_KEY_ID=...
AWS_SECRET_ACCESS_KEY=...
AWS_S3_BUCKET=...
EMAIL_USER=...
EMAIL_PASS=...
GOOGLE_MAPS_API_KEY=...
```

### Auto-Generated Variables
```

JWT_SECRET=... (auto-generated by Render)
```

## 📝 Post-Deployment Steps
